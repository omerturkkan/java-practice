# java-practice

Personal Java practice repository. Daily 3-5 exercises covering core Java topics.

## Package Structure

| Package | Topics |
|---|---|
| `basics` | variables, datatypes, operators, strings, scanner |
| `controlflow` | conditionals, loops |
| `methods` | overloading, recursion, varargs |
| `arrays` | single-dim, multi-dim, sorting |
| `oop` | encapsulation, inheritance, polymorphism, abstraction |
| `interfaces` | default methods, functional interfaces |
| `exceptions` | checked, unchecked, custom |
| `collections` | list, set, map, queue, deque |
| `generics` | generic class/method, bounded, wildcard |
| `functional` | lambda, method reference, optional, streams |
| `datetime` | LocalDate, LocalDateTime, formatting |
| `fileio` | reading, writing, NIO |
| `multithreading` | threads, runnable, synchronization, executor |
| `designpatterns` | creational, structural, behavioral |
| `testing` | JUnit 5, assertions |
